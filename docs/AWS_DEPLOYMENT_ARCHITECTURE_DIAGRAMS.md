### Network Flow Explanation

1. **Internet Gateway (IGW):** Entry point for all traffic into VPC
2. **Public Subnets:** Subnets with route to IGW (can access internet)
3. **Multi-AZ:** Tasks spread across 2 availability zones for high availability
4. **ALB:** Distributes traffic across healthy ECS tasks
5. **Security Groups:** Act as virtual firewalls
